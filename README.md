# Fancim project

일리오의 팬심 M 마케팅을 위한 데이터 분석 및 채팅 데이터 감성 분석 기업 협업 프로젝트

## 기업 소개 

셀럽들의 니즈를 해결하기 위해 팬심에서는 개인정보 노출 없이 셀럽과 팬이 선물을 주고 받을 수 있는 선물후원,  셀럽에게 원하는 콘텐츠를 요청하는 랜선 미션, 셀럽과 팬이 직접적으로 대화를 할 수 있는 메신저 등 3가지 기능을 제공.
